Hyperlinks (a element)

Op Twitter was er een kleine poll bij front-end developers met de vraag welk HTML-element zij het belangrijkst vonden. De duidelijke winnaar was: a

  • Het href attribuut;
  • linken naar een andere site;
  • linken naar een andere pagina (lokaal)
  • links binnen een pagina

Video Kevin Powell

In zijn inleidende cursus HTML en CSS heeft Kevin Powell het over het a element. Hij behandelt linken naar een andere site en naar een lokaal bestand. In dit filmpje codeert hij niet in Visual Studio Code, maar in Atom (VS Code bestond nog niet toen deze cursus gemaakt werd, maar zoals gezegd maakt dat eigenlijk niets uit).

Linken binnen een pagina

Op een lange pagina kan het nuttig zijn om naar bepaalde delen van de pagina te linken. Dat doe je door gebruik te maken van een id attribuut (voor het doel) en deze string te laten voorafgaan door een # in het href attribuut van het a element, zoals in volgend voorbeeld getoond wordt.

<p><a href="#validatie">Spring naar de sectie over validatie.</a></p>
<p>lange tekst hier …</p>
…

<h2 id="validatie">Validatie</h2>
<p>Code die je net hebt geüpload op de server laat je 
altijd valideren. Dat kost weinig moeite en …</p>
